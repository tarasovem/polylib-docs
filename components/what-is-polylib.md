# Что такое Polylib

Polylib - минималистичная библиотека для создания Web Components с декларативным подходом к шаблонам и данным.

## Зачем использовать Polylib

Polylib помогает сфокусироваться на логике компонента:

- упрощает работу с реактивными свойствами;
- дает удобные шаблонные конструкции для рендера;
- делает работу со стилями и инкапсуляцией более предсказуемой.

## Ключевые принципы

- **Компонент как изолированная единица**: состояние и поведение живут внутри элемента.
- **Данные через свойства**: источник правды находится в свойствах компонента.
- **Декларативный UI**: шаблон описывает, как должен выглядеть интерфейс при текущем состоянии.

## Минимальная ментальная модель

1. Вы объявляете свойства компонента.
2. Шаблон читает эти свойства.
3. При изменении свойств интерфейс обновляется.

## Когда Polylib подходит лучше всего

- когда вы строите дизайн-систему на Web Components;
- когда нужна переиспользуемость между проектами и фреймворками;
- когда важны инкапсуляция и контролируемая стилизация.
